# ⚖️ Balancing Life To-Do-List (2021-2 OSP)
React-Native / expo로 구현한 To Do List.  
일과 관련된 To-Do-Item을 Work로, 삶과 관련된 To-Do-Item을 Life Category로 설정하여  
Work와 Life의 균형을 progress Bar로 확인할 수 있다.
